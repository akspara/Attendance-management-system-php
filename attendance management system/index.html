LANGUAGES USED

PHP

PHP(Hyper Text Preprocessor) is used here to provide the backend of the website.Mainly it is used to connect to the database and store the information of the user by marking the attendance on a daily basis.

CSS

CSS is used to provide the design of the website and to make it user friendly and responsive. All the designs including login form,front page,admin page,display,headers,footers are created using CSS.

JS

Javascript is used is the least amount as all the backend is provided by PHP in this website. Still some function like sliding the pictures in front page and show password in login page is created using javascript.

ABOUT THE WEBSITE



The sole purpose of the website is to provide a better management of the student record which are small in size. Thes website is efficient for internship programs and training programs which does not need big data values to store the records.The working of the web app is simply same as DBMS project, its just providing you the interface of the DBMS to store data easily. The user needs to signin the first in order to get the details of his/her attendance. If he/she is already not registered to Database then they need to register first before using the management system. Once the detail is provided you can access your details where you can mark present of that particular data.Once the present is marked , you cannot change it directly from this website.If your opening your website after more than one day, after marking oresent it will automatically fill absent for the days you didn't mark present.
